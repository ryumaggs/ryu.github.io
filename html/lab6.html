Accessing Characters in a String

  • x = "CS 105 is awesome"

  • x[0]
  • x[-1]
  • x[0:6]
  • x[10:17] + x[6:10] + x[0:6]
  • x[:4]
  • x[5:]

If Statements

    <statements>

    <statements>

    <statements>
