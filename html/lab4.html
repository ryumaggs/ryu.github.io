IDLE - Integrated DeveLopment Environment

IDLE is a tool for interacting with Python and writing Python programs. IDLE is convenient for a number of reasons:

  • it gives the appropriate indentification
  • it uses colors for reserved words, "strings" etc..
  • it is easy to create a .py file
  • it runs on your own machine, can be downloaded from http://www.python.org/download/

Let's take a tour: Python is installed in our lab (Windows -> All Programs -> Python3.3 -> IDLE).



Python Basics

Try out the following commands and see what happens:

  • General
    • print("Hello, World!")
    • #print("Hello, World!")
      • Commands that start with a "#" are comments
      • Comments are ignored by the interpreter
      • Comments should briefly explain what is going on, but only when necessary
  • Assignment
    • x = 5
      • "x" is a variable, and we are assigning it a value of 5.
    • x=5
      • Note: whitespace in code doesn't change, but can make your code more readable
    • my_variable = 12
      • Variable names can be whatever we like
      • ProTip: Try to keep variable names relevant to the task at hand
  • Comparison
    • x > 5
    • x == 5
    • x == 6 or x > 8
      • Note: like in SQL you can use the words "and" and "or"
      • Note: unlike in SQL, "and" and "or" must be lowercase
  • Input
    • z = input("Enter a number: ")
    • print(z)
    • z = eval(input("Enter a number: "))
    • print(z)
      • What is the difference?
  • String Contatenation
    • print("Hello " + "World")
    • word2 = "Back!"
    • print("Hi " + word2)

Python Programs

To write and test the program, do the following:

  1. Create a new program file in IDLE (File->New)
  2. Write the program that
    • Asks the user what their name is
    • Greets the user with "Hello <entered name> !"
  3. Save the program as hello.py.
  4. Run it by using Run -> Run Module or the F5 key.

  5. Example execution:
      Please enter your name: John
      Hello John!
